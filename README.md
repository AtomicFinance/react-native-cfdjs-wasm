# react-native-cfdjs-wasm

WebView wrapper to run cfdjs-wasm in React Native.

## Installation

```sh
npm install react-native-cfdjs-wasm
```

## Usage

```js
import { initCfd } from "react-native-cfdjs-wasm";

// ...

const cfd = await initCfd();
```
